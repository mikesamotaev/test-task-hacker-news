# About the app

This is a react app for fetching and displaying limited amount of random hacker news. The app fetches and gathers ordered data using endpoints of API https://github.com/HackerNews/API


# How to launch this app

## Clone it using remote repo
git clone  https://github.com/MikeDevDesigner/hacker-news-app.git

## Install dependencies
* cd hacker-news-app
* npm install

## Run tests/development version/build version
* npm run test (tests)
* npm run start (development version)
* npm run build (build version)
